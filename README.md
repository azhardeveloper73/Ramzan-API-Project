# 🌙 Ramzan API Project

![Ramzan Banner](https://raw.githubusercontent.com/azhardeveloper73/Ramzan-API-Project/main/assets/ramadan-banner.png)

A complete **Ramzan Ramadan website** showcasing daily Ramadan calendar, Naat, Quran Pak links, and About Us page. Fully responsive with interactive navbar and active page highlighting.  

---

Check the live website here: [Ramzan Website Live](https://azhardeveloper73.github.io/Ramzan-API-Project/)

---

## 📂 Repository
Explore the source code on GitHub: [Ramzan API Project Repo](https://github.com/azhardeveloper73/Ramzan-API-Project)

---

## 💻 Features

- **Dynamic Ramadan Calendar** fetched using [Aladhan API](https://aladhan.com/prayer-times-api)
- **Responsive Design** for Desktop and Mobile
- **Interactive Navbar** with active page highlighting
- **Home, Naat, Quran Pak, and About Us pages**
- **Fav Icon & Thumbnails** included
- Smooth **Hover Effects** on navbar links
- Ready for deployment on **GitHub Pages** or any web server

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Responsive Design)
- JavaScript (DOM Manipulation, API Fetch)
- Git & GitHub for version control
